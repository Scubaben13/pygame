# Jogatina — Jogo de Atenção

Um jogo de ritmo e reação com clima de "jogatina de bar".

## Como Jogar
- **Jogo Simples**: Pressione `ENTER` quando sua carta-alvo aparecer.
- **Jogo Duplo**: 
  - Jogador 1 (esquerda): `ESPAÇO`
  - Jogador 2 (direita): `ENTER`
  - Turnos alternados. Quem acertar mais rápido leva o ponto!

## Instalação
1. Instale o Python 3.8+
2. Execute:  
   ```bash
   pip install -r requirements.txt
   python main.py
